# Credit Card Fraud Detection System using Taipy


## Introduction 
With the rapid growth of online payments and digital banking, credit cards have become one of the most widely used financial tools. However, this growth has also increased the risk of fraudulent transactions. Credit card fraud occurs when unauthorized users gain access to card information and perform transactions without the owner's consent. Detecting fraudulent transactions quickly is essential for financial institutions to prevent financial losses and protect customers. Machine learning techniques can help identify suspicious transaction patterns and detect fraud effectively. This project proposes a fraud detection system with an interactive interface built using Taipy.
